# AI4ALPHA

Financial Event Timeline Intelligence Platform

## Overview

AI4ALPHA transforms scattered financial data (news, SEC/SEDAR filings, market data) into structured hourly event timelines. Built for quantitative research, algorithmic trading, and LLM agents.

## Features

- Multi-Source Data Fusion
- Hourly Event Buckets
- AI-Powered Extraction
- Quant-Ready Features
- API & MCP Ready

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Shadcn/UI

## Development

```bash
npm install
npm run dev
```

## Deployment

Deploy to Vercel with one click.
